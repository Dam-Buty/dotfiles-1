# dotfiles

## Install

```bash
make install
```

## Structure

```
.
├── assets            # static assets
│   ├── fonts
│   └── images
│
├── bin               # user-defined commands (the folder
│                     # is added to $PATH)
│
├── colors            # theme colors
│
├── config            # bunch of configs, including awesome wm config
│   ├── awesome
│   └── ...etc.
│
└── dot               # actually dotfiles.
    ├── gitconfig
    ├── profile
    ├── vimrc
    ├── zshrc
    └── ...etc.
```
